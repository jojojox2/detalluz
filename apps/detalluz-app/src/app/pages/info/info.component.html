This application was designed only for personal use.

The scope of it is limited only to already registered users for the i-DE company (electrical distributor in Spain), with no relations or vinculations whatsover with this company.
The credentials used in this application are only to login to these external services, and are not stored in any server, database or similar storage, beyond the session in your own private browser.
The information retrieved are limited to the hourly consumption as detected by your electricity meter.

The hourly pricing used in this application is retrieved online from the Red Eléctrica de España website.

The source code for this application can be found in GitHub.
